Die "Baden-Umleitung"

Fährt ein CJX-Zug auf der Pendlerstrecke aufgrund von Baustellen oder Störungen eine alternative Route, entfällt in der Regel der Halt in Baden bei Wien. Das System registriert eine Umleitung automatisch, wenn für eine Zugnummer Ankünfte/Abfahrten in Ternitz und Wien Meidling vorliegen, das Ereignisprotokoll in Baden jedoch fehlt.
